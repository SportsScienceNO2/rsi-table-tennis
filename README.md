# The Role of RSI in Table Tennis: Olympic & Paralympic Analysis

This repository contains the full dataset, source code, and statistical reports for the study: **"The Role of the Reactive Strength Index in Talent Identification and Training Prescription for Youth Olympic and Paralympic Table Tennis Players"**.

## ðŸ’» 2. Reproducibility Pipeline

To replicate the study, run the Jupyter Notebooks in the `notebooks/` folder in the following **exact order**:

### **Step 1: Data Preparation & Cleaning**
- **Notebook:** `CLEANING.ipynb`
- **Goal:** Standardize column headers (Portuguese to English), handle missing values (NaN conversion), enforce variable types (float), and ensure privacy by dropping non-essential identifiers.
- **Input:** `Data_TableTennis.csv`
- **Output:** `Data_TableTennis_Ready.csv`

### **Step 2: Statistical Analysis & Modeling**
- **Notebook:** `MAIN_ANALYSIS.ipynb`
- **Goal:** 1. Compute **Reliability** (ICC & CV%) and **Normality** (Shapiro-Wilk).
    2. Generate **Table 1** (Descriptive Characteristics).
    3. Perform **RSI Analysis** (RM-ANOVA for drop heights & Inter-group T-tests).
    4. Calculate **Spearman Correlations** and **Linear Regression Models**.
    5. Conduct Post-hoc **Sensitivity Power Analysis** (G*Power logic).
- **Input:** `Data_TableTennis.csv` (or `Data_TableTennis_Ready.csv`)
- **Output:** Statistical Reports (Text) and Tables for the manuscript.

## ðŸ“Š 3. Key Findings & Statistical Outputs

The statistical outputs used in the manuscript are generated by the `MAIN_ANALYSIS.ipynb` notebook.

### **Reliability & Normality Report**
- **Output:** Console Print Block 1.
- **Metric:** Intraclass Correlation Coefficient (ICC) and Coefficient of Variation (CV%).
- **Insight:** Confirms that 10m sprints and Jump tests have "Excellent" reliability (>0.90), while RSI at 45cm showed lower precision.

### **RSI Optimization Analysis (ANOVA)**
- **Output:** "RSI Analysis Master Report".
- **Test:** Repeated Measures ANOVA (15cm vs 30cm vs 45cm).
- **Insight:** Identifies 30cm as the optimal drop height for both groups, despite the lack of statistical significance in the omnibus test.

### **Predictive Modeling (Regression)**
- **Output:** "Simple Linear Regression Analysis".
- **Test:** Linear Regression (R2, F-test, Unstandardized beta).
- **Insight:** * **Conventional:** 10m Sprint explains **36%** of Ball Velocity (p < 0.01).
    * **Para-athlete:** RSI at 30cm explains **26%** of Technical Efficiency (Accuracy).

### **Sensitivity Power Analysis**
- **Output:** "Post-Hoc Statistical Power Analysis Report".
- **Metric:** Achieved Power (1 - beta) and Sensitivity (d).
- **Insight:** Demonstrates robust power (80.7%) for the main group comparison but reduced sensitivity (42.7%) for the Paralympic regression model.
